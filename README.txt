# Gen Z Email Cards

This project is a single-page web app for managing and sharing a collection of Scribd embed links, organized as visually appealing cards. It is designed for Gen Z users and features a modern, colorful UI with theme switching, search, and sharing/import functionality.

## Features

- **Add Embeds:** Add new Scribd embed links with a title and category.
- **Search:** Instantly filter cards by category or title.
- **Themes:** Switch between Bubblegum, Matrix, and Vaporwave themes.
- **Modal Preview:** Click a card to preview the embed in a modal dialog.
- **Share/Import:** Export your custom embeds as JSON, or import shared embeds from others.
- **Persistence:** All embeds are saved in your browser's localStorage.

## Usage

1. **Open `email.html` in your browser.**
2. Use the controls at the top to add new embeds.
3. Search for cards by typing in the search box.
4. Click a card to view it in a modal.
5. Use the "Share" button to export or import embed collections.
6. Change the theme using the dropdown.
