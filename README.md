<h1 style="font-size:3em; margin-bottom:0.2em;">Gen Z Email Cards</h1>

<p>
  <b>Modern, colorful web app for managing and sharing Scribd embed links.</b>
</p>

<hr>

<h2>Features</h2>
<ul>
  <li><b>Add Embeds:</b> Add new Scribd embed links with a title and category.</li>
  <li><b>Search:</b> Instantly filter cards by category or title.</li>
  <li><b>Themes:</b> Switch between Bubblegum, Matrix, and Vaporwave themes.</li>
  <li><b>Modal Preview:</b> Click a card to preview the embed in a modal dialog.</li>
  <li><b>Share/Import:</b> Export your custom embeds as JSON, or import shared embeds from others.</li>
  <li><b>Persistence:</b> All embeds are saved in your browser's localStorage.</li>
</ul>

<h2>Usage</h2>
<ol>
  <li>Open <code>email.html</code> in your browser.</li>
  <li>Use the controls at the top to add new embeds.</li>
  <li>Search for cards by typing in the search box.</li>
  <li>Click a card to view it in a modal.</li>
  <li>Use the "Share" button to export or import embed collections.</li>
  <li>Change the theme using the dropdown.</li>
</ol>

<h2>Notes</h2>
<ul>
  <li>Only valid Scribd embed URLs are accepted.</li>
  <li>Your custom embeds are stored locally in your browser and are not shared unless you export them.</li>
  <li>The initial set of embeds is included for demonstration.</li>
</ul>
